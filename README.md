# Rate Repositry App 

Full Stack Open part 10. Focusing on React Native and it's corresponding FE technologies.  

Backend used: https://github.com/fullstack-hy2020/rate-repository-api 

## Technologies used:
- React Native
- Apollo/Client
- GraphQL
- Expo
- Formik
- Yup
- Jest
